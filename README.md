## ğŸŒ©ï¸ First Thought Principle
> *"Why do we build? To bridge the gap between utility and art."*

**Mitti** is not just an e-commerce UI; it is a digital tribute to 5,000 years of pottery tradition. The goal was to create a web experience that feels as organic and tactile as the clay itself, while maintaining the speed and responsiveness required by modern users.

## âœ¨ Key Features

### ğŸ¨ Aesthetic Engineering
- **Earthy Palette:** A carefully curated color system (`#E8DCC8`, `#5D4037`) that mimics sun-dried clay and river sand.
- **Tactile UI:** Deep, solid shadows (`box-shadow: 15px 15px 0 #d7ccc8`) create a physical, card-like depth rarely seen in flat design.
- **Typography:** Uses *Georgia* serif to evoke a sense of history and storytelling.

### ğŸ› ï¸ Technical Implementation
- **Responsive Grid:** Migrated from legacy `inline-block` layouts to a robust **Flexbox** architecture, ensuring perfect alignment on all viewports.
- **Local Asset Management:** Implemented a self-hosted HTML5 video player to eliminate external dependencies (YouTube API errors) and ensure zero-latency playback.
- **DRY Codebase:** Modularized styles into `common.css` for global resets and utility classes, keeping page-specific stylesheets lightweight.
